## 📋 Descripción

Aplicación web desarrollada con **Spring Boot (Java)** que demuestra mis habilidades en desarrollo backend, contenerización con Docker y buenas prácticas de control de versiones con Git y GitHub.

Este proyecto forma parte de mi portafolio como **Aprendiz SENA en Análisis y Desarrollo de Software** y refleja mi capacidad para crear aplicaciones modernas, escalables y listas para producción.
